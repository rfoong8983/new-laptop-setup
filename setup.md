## Escape Sequences

| shortcut |  action  | Esc+ |
| cmd + <- | send esc |  OH  |
| cmd + -> | send esc |  OF  |
| opt + <- | send esc |  b   |
| opt + <- | send esc |  f   |

### For option delete
1. Go to preferences cmd + ,
1. Profiles
1. Keys
1. Set 'Left Option key' to 'Esc+'

## Open tab in curr dir
